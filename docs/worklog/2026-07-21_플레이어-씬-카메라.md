### 2) 임시 비주얼 + 조준 레이저
> "2D 탑뷰에 맞게 메인카메라를 플레이어 위치에 설정, 임시로 플레이어를 빨간 구체로, 사격 시 가리키는 방향을 볼 수 있는 레이저 구현."

- 메인 카메라: **Orthographic 전환**, size 6, 위치 `(0,-4,-10)` (초기엔 플레이어 위치)
- 플레이어 임시 비주얼: `Player/Visual` **빨간 구체**(Sphere) + `Assets/Materials/PlayerRed.mat`(URP Unlit) / 3D 콜라이더 제거
- 조준 레이저: `PlayerShooter`에 `LineRenderer` 레이저 사이트 추가
  - 항상 마우스 방향 표시, **발사 순간 굵어졌다 복귀**, 끝으로 갈수록 옅어짐(`Sprites/Default` 셰이더로 URP 매젠타 방지)

### 3) 카메라 팬(WASD)
> "카메라는 플레이어에 고정되면 안 되고, WASD로 탑뷰 각도를 고정한 채 화면 이동."

- `Assets/Scripts/CameraPanController.cs` (신규) — Main Camera에 부착
  - WASD/방향키로 XY 평면 이동, **회전(탑뷰) 고정**, Z 고정
  - `_panSpeed`, 선택적 이동 범위 제한(`_useBounds`)

### 4) 마우스 휠 줌
> "마우스 휠로 확대/축소 가능하게."

- `CameraPanController`에 줌 추가: 휠 위=확대(size↓)/아래=축소(size↑), `_zoomSpeed`, `_minZoom`(2)~`_maxZoom`(15) clamp, 직교 카메라 전용

## 테스트 방법 (Stage 씬)
1. Play → **WASD** 화면 이동(탑뷰 유지), **마우스 휠** 확대/축소
2. **마우스 이동** 시 빨간 레이저가 조준 방향 표시 → **좌클릭** 시 레이저 강조 + Console `발사! 방향=…, 남은 탄환=…`, 탄환 0에서 "탄환 없음"
3. GameManager 컴포넌트 우클릭 → `Debug/적 처치·민간인·플레이어 사망` 으로 클리어/실패 + 씬 전환 확인
